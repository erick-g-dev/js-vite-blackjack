# Blackjack - Vite Project 🃏



Proyecto de **Blackjack** desarrollado con **JavaScript**, usando **Vite** para un entorno moderno y rápido de desarrollo frontend.

---

## Demo
Si quieres probarlo en vivo, abre tu navegador después de correr el proyecto localmente.  

---

## Tecnologías
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite

---

## Instalación y ejecución
1. Clona el repositorio:

```bash
git clone https://github.com/erick-g-dev/js-vite-blackjack.git

3. Entra a la carpeta del proyecto:
cd js-vite-blackjack

4. Instala dependencias:
npm install

5. Corre el proyecto:
npm run dev

Abre tu navegador en la URL que Vite te indique (por lo general http://localhost:5173).
